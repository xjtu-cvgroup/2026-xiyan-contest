# V3.97 隐藏地图混合策略

分支：`exp/v3.97-hidden-map-hybrid`

## 唯一目标

1. 我方可交付是硬约束。
2. 有严格证明时阻止对手交付。
3. 无法证明封锁时，由 Planner 最大化最终分数。

三套策略不同时竞价。`HybridStrategy` 每帧只让一个模式掌权：

- `PRIMARY_WARDEN`：删除某个 `KEY_PASS` 后，起点无法到达 S14；完整运行
  V3.96.34 Warden。
- `SCORE_RACE`：不存在必经 `KEY_PASS`；完整运行 V3.56 Planner。
- `GATE_WARDEN`：Planner 后段严格取得 S14 控制权后，粘性切入 Warden。

## S14 接管硬门

必须同时满足：

- 对手尚未到达或验核 S14；
- 所有正常 S14 入边在疾行速度下仍至少需要 5 帧；
- 删除 S14 后不能先到 S15 再按特殊返回规则反穿宫门卡；
- 我方保守 ETA 至少领先对手乐观 ETA 8 帧；
- 任务分达到 120，或已到 350 帧/RUSH，或对手距宫门不超过 120 帧；
- 我方至少 7 篓好果，并能在交付需求外保留 10 帧余量。

切入后不可返回 Planner。S14 反应设卡若会吃掉我方交付余量，则直接验核/
交付，不为最后一张卡把自己拖死。

## 回归纪律

- 公开图必须识别 S10 为必经点，整局结果保持 V3.96.34。
- 添加 `S09 -> S11` 旁路后必须禁用 S10 Warden。
- S14 入边不足 5 帧、可绕到 S15、先手不足或交付余量不足时不得接管。
- 快速闸门：`client/selftest.py` 与 `client/warden_scenarios.py`。
